<h3>Deskripsi : </h3>
<p>GitHub repository Foenyms adalah kumpulan script game untuk Unity dengan tema puzzle dan teka-teki yang unik. Terdapat berbagai fitur seperti kontrol karakter, animasi, interaksi objek, dan UI. Cocok untuk pengembang game yang ingin membuat game dengan gameplay yang menarik dan menantang. Terdapat dokumentasi dan tutorial yang mudah diikuti.</p>
<br>
<img src="https://tse3.mm.bing.net/th?id=OIP.LqkJG4bURNRDiqDx096VDAHaEH&pid=Api&P=0"/>
<br>
<p>Import script ke dalam proyek Unity.
Tambahkan GameObject karakter ke dalam scene.
Tambahkan Character Controller component pada GameObject karakter.
Tambahkan Animator component pada GameObject karakter.
Atur nilai-nilai variabel pada script sesuai kebutuhan, seperti _walkSpeed, _runningSpeed, dan jumpHeight.
Untuk menggunakan movement mode Platformer, pastikan variabel _movementMode pada script diubah menjadi "Platformer".
Jika menggunakan mode Strafe, pastikan GameObject kamera dihubungkan dengan variabel _cameraRig pada script.
Atur animasi karakter sesuai dengan kondisi yang diinginkan, seperti animasi berjalan atau berlari.
Jalankan game dan coba gerakkan karakter menggunakan tombol keyboard atau joystick sesuai dengan mode yang dipilih.
Untuk detail lebih lanjut, silakan lihat komentar pada script Character Movement.</p>
